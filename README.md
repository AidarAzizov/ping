# PING on Linux

Реализация системной комманды ping

Реализованные флаги:
	
	-dt (задержка между пингами);
	-c (количество повторений пинга);
	-h (вывод информации о флагах);
	-s (размер пакета);
	-t (определяет TTL);
	-w (временной тайм-аут для выполнения программы);
	
Сборка:

	Запуск скрипта mymake
